# 全体設計

## システム概要

座席レイアウトの区分管理とチケット番号自動採番を行うデスクトップアプリ。
Excelで定義された座席レイアウトをUIに表示し、座席ごとに区分（色）を割り当て、チケット番号を自動採番する。

### 利用シナリオ

- ユーザーは2名、リモートで独立して使用（同時編集なし）
- 状態はJSONファイルにエクスポートし、相手にファイル送付することで共有
- Mac・Windows両対応

## JSONスキーマ（共有ファイル）

```json
{
  "version": 1,
  "exportedAt": "2024-01-01T00:00:00.000Z",
  "categories": [
    { "id": "A", "name": "A区分", "color": "#e74c3c" }
  ],
  "layout": [
    ["A1", "A2", "A3", null, "A4", "A5", "A6", "A7", null, "A8", "A9", "A10"]
  ],
  "seats": {
    "A1": { "categoryId": "A", "ticketNo": "A001" }
  },
  "ticketConfig": {
    "prefix": "A",
    "startNum": 1
  }
}
```

**`layout`**: 2D配列。`null` = 通路（空セル）、文字列 = 座席ID。行・列の物理的な位置を保持する。

**`seats`**: 座席IDをキーとする辞書。区分未割当の座席はキーが存在しない。`ticketNo` は採番前 `null`。

## Electron統合方針（フェーズ4）

- Pythonプロセスを子プロセスとして起動し、ローカルHTTPサーバー（localhost）として動かす
- フロントエンドJSはフェーズ1の実装をそのまま流用し、Excelインポート・出力ボタンのみPython APIを叩くよう追記
- `phase1_ui/` のHTML/JS/CSSはElectron側のBrowserWindowでそのままロードする
