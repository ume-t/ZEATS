# ZEATS

座席レイアウトの区分管理とチケット番号自動採番を行うデスクトップアプリ。

## 名前の由来

**ZEATS** は **Zone** と **Seats** を組み合わせた造語です。

座席（Seats）をゾーン（Zone）ごとに色分けして管理するというアプリの核となる機能を、短く表現しています。

---

## 起動方法

### フロントエンド（ブラウザ）

`phase1_ui/index.html` をブラウザで直接開くだけで起動します。

```bash
open phase1_ui/index.html        # macOS
start phase1_ui/index.html       # Windows
```

インストール不要・サーバー不要です。

---

## 基本的な使い方

### Excelファイルから座席データを読み込む

**Step 1 — ExcelをJSONに変換する（Python）**

```bash
conda activate zeats
cd phase2_python
python excel_importer.py ../tmp/大阪フォーマット.xlsx ../output.json
```

`output.json` が生成されます。

**Step 2 — JSONをブラウザに読み込む**

ブラウザで `phase1_ui/index.html` を開き、
ヘッダーの **「Excelから変換したJSONを読込」** ボタンから `output.json` を選択します。

シートタブが表示され、ブロック単位で座席が表示されます。

---

## 操作方法

### 区分の割り当て

1. ヘッダーの区分ボタン（楽天／ぴあ／KKDAY など）をクリックして選択
2. 座席をクリック → 選択中の区分が割り当てられ、色が付く
3. **消去** ボタンを選択した状態でクリック → 割り当てを外す
4. 座席を右クリック → コンテキストメニューから区分を直接変更

### ブロック検索（Excelインポート時）

検索ボックスにブロック名の一部を入力するとフィルタリングされます。
例：`赤ゾーン31` と入力 → 「赤ゾーン31ブロック」だけが表示されます。

### チケット番号の採番

1. 右サイドバーの「チケット番号」パネルでプレフィックスと開始番号を設定
2. **番号を割り振る** をクリック → 区分割当済みの全座席に連番が付与される
3. 採番順：ブロック → 列 → 座席番号の順（上から下・左から右）

### データの保存と共有

| 操作 | ボタン | 説明 |
|------|--------|------|
| 現在の作業を保存 | **JSON出力** | 全シートの区分・チケット番号を含むJSONを保存 |
| 保存した作業を再開 | **作業JSONを読込** | 前回出力したJSONを読み込んで再開 |
| Excelデータを更新 | **Excelから変換したJSONを読込** | Pythonで再変換したJSONを読み込む |
